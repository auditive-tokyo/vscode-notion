# Development

## Setup

```bash
git clone https://github.com/auditive-tokyo/Notion-VSCode.git
cd Notion-VSCode
npm install
npm run build
```

## Run in Development

```bash
code --extensionDevelopmentPath=.
```

## Tests and Quality

```bash
npm run lint
npm run format
npm run check
```

For contributing guidelines, see ../CONTRIBUTING.md.
