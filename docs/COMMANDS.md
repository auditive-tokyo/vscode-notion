# Commands

Configuration is done via VS Code Settings (Cmd+,).

| Command                | Description                                     |
| ---------------------- | ----------------------------------------------- |
| `Notion: Open Page`    | Open a page by URL or ID                        |
| `Notion: Refresh Page` | Refresh the current page view                   |
| `Notion: Clear Cache`  | Clear cached pages (useful if content is stale) |

Run `Notion:` in the command palette to see all available commands.
