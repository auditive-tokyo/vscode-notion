# Features

- Browse Notion pages in a hierarchical tree
- Full Markdown rendering (headings, lists, code blocks, callouts)
- Inline databases with multiple view modes:
  - Table
  - Calendar (date-based)
  - Timeline (date ranges)
  - Board (status-based)
- Mermaid diagram rendering
- Light and dark theme support
- Official Notion API integration (no scraping)
